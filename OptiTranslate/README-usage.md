Usage notes:
- Press Option+Space to translate the current selection. The app simulates Cmd+C to copy selection and then translates it. Make sure clipboard access is allowed.
- To supply OPENAI_API_KEY, set it in the environment (e.g., export OPENAI_API_KEY=sk-...) or configure the Run scheme in Xcode.
- Translations are appended to ~/Documents/Translations.md with date, original text and Chinese translation.
